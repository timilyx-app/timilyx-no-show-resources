## Quick start

1. Browse the folders below.
2. Copy the template text you need.
3. Replace placeholders like: `{client_name}`, `{date}`, `{time}`, `{policy_url}`.
4. Use it in WhatsApp, email, SMS, or your booking system.

## Folder structure

- `/no-show-policy` → policy templates
- `/whatsapp-templates` → WhatsApp reminder & confirmation copy
- `/email-templates` → email confirmation & reminder copy
- `/sms-templates` → short versions for SMS
- `/checklists` → operational checklists to reduce no-shows

## Best practices (recommended)

- Send **confirmation** right after booking
- Send **reminders** (24h + 2h) depending on your audience
- Make cancellation/rescheduling easy and clear
- Use a clear policy (late cancellation / no-show rules)
